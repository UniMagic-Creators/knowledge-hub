UnityのProjectウインドウで右クリックし、「Create」→「U# Script」をクリックしてください。

すると、エクスプローラーの画面が開くので、ファイル名を指定し、「保存」をクリックします。

今回は、「MyFirstUdon」としました。

以下のように、2つのファイルが生成されればOKです。
